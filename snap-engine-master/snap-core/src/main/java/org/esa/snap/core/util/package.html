Provides several utility classes and sub-packages used throughout the MERIS/(A)ATSR Toolbox software.

Currently the following subpackages exist (sorted by functionality):

  • {@link org.esa.snap.util.geotiff Support classes for GeoTiff file handling}
  • {@link org.esa.snap.util.io I/O-Utilities}
  • {@link org.esa.snap.util.jai Support classes for JAI} (Java Advanced Imaging API)
  • {@link org.esa.snap.util.logging Logging utilities}
  • {@link org.esa.snap.util.math A collection of mathematics routines and classes}
