This package contains interfaces and classes used for the implemenation of cartographic map projections. The package has three very important classes.
  • The class {@link org.esa.snap.core.dataop.maptransf.MapProjection} is used to represent a specific parameterized prjection.
  • the interface {@link org.esa.snap.core.dataop.maptransf.MapTransform} provides the mathematical algorithm for the transformation and
  • the {@link org.esa.snap.core.dataop.maptransf.MapInfo} class holds information required to bring the map co-ordinate system to an image raster co-ordinate system.
